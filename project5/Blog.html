Important FAQ.....

1. What is the Document Object mMde (DOM) ?

The Document Object Model (DOM) is a programming interface that represents the structure of a web page as a tree, allowing scripts to dynamically access and manipulate its elements and content.

2. How do you select an element from the Dom ?

Use methods like getElementById(), querySelector(), getElementsByClassName(), or getElementsByTagName().

3. What is even delegation in the context of the DOM, and Why is it useful ?

Event delegation uses a parent element's listener to handle child events via event bubbling, saving resources and supporting dynamic elements.

4. How do you manipulate an element's attributes and styles using thye Dom ?

Use setAttribute(), getAttribute(), removeAttribute() for attributes, style for inline styles, and classList for managing CSS classes.
